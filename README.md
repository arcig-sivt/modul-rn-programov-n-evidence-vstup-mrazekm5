## Popis

Naprogramujte modul pro evidenci vstupu a výstupu osob pomocí čipových karet.

## Požadavky

### 1. Evidence příchodů a odchodů

Ze vstupního senzoru přichází pouze **ID karty**. API musí zaznamenat příchod i odchod spolu s časovým razítkem.

### 2. Historie

Modul si pamatuje historii všech zaznamenaných příchodů a odchodů. Historii uložte tak, aby šla procházet a filtrovat.

### 3. Provázání ID karty s osobou

Každou kartu lze přiřadit konkrétní osobě. Promyslete jaké to má implikace.

### 4. Složitější use-case

Vymyslete si co chcete pokytnout za zajímavý dotaz.
